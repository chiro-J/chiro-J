- ✅ 핵심 기능1: 반응형 웹 레이아웃 구현

## 💡 주요 기능 설명

### ✨ 기능 1: 외부 컴포넌트를 이용한 이미지 데이터 출력

* `Swiper.js` 를 활용하여 이미지 갤러리 구현

### 🌦️ 기능 2: 외부 API 연동

* OpenWeatherMap / Kakao API 등 활용
* Fetch / Axios 등으로 비동기 통신 구현

### 🔐 기능 3: 사용자 인증 처리

* JWT 토큰 발급 및 검증
* 로그인 상태 저장 (localStorage 등)
